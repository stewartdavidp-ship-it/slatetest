Slate v1.0.8
============
Release Date: January 23, 2025

PWA SAFE AREA SUPPORT
---------------------
Added iOS safe area inset support for PWA mode:
- Body respects notch/Dynamic Island spacing
- Content doesn't get hidden behind system UI
- Better experience on modern iPhones

This ensures the game displays properly when installed
as a PWA on devices with notches or Dynamic Island.
